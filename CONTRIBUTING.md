# Contributing

Thanks for helping improve Awesome Word Learning.

## What Belongs Here

This list is for resources that help people learn vocabulary or word usage:

- Vocabulary-by-topic tools.
- Spaced repetition and flashcard tools.
- Reading and word mining tools.
- Dictionaries, pronunciation tools, and example sentence resources.
- High-quality language-specific vocabulary resources.

## Quality Bar

Please add resources that are:

- Useful within the first few minutes.
- Relevant to vocabulary or word learning.
- Publicly accessible.
- Stable and not obviously abandoned.
- Free, freemium, open-source, or clearly priced.

Avoid:

- Link farms.
- Thin affiliate pages.
- Aggressive ads or misleading downloads.
- Duplicate entries.
- Purely promotional submissions without clear learner value.

## Entry Style

Use this format:

```md
- [Name](https://example.com/) - One short sentence explaining why it helps vocabulary learning.
```

Keep descriptions neutral and concise.

## First-Party Resources

This repository is maintained by the WordMeadow team. WordMeadow entries are allowed, but they should stay transparent and should not crowd out other resources.

## Pull Requests

1. Check that the resource is not already listed.
2. Add it to the most relevant section.
3. Keep the description to one sentence.
4. Open a pull request explaining why the resource is useful.
