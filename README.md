# Awesome Word Learning

A curated list of tools and resources for learning vocabulary, building word
knowledge, and reviewing words by topic, context, sound, and usage.

The focus is practical word learning: vocabulary by theme, spaced repetition,
reading-based word mining, pronunciation, dictionaries, and multilingual
learning resources.

## Core Tools

- [Anki](https://apps.ankiweb.net/) - Flexible spaced-repetition flashcard software with a large ecosystem of shared decks.
- [Clozemaster](https://www.clozemaster.com/) - Cloze-based vocabulary practice in sentence context.
- [Memrise](https://www.memrise.com/) - Vocabulary courses with memory prompts and review.
- [Quizlet](https://quizlet.com/) - Flashcards and simple study modes for custom vocabulary lists.
- [Readlang](https://readlang.com/) - Read web pages, translate words, and turn lookups into review items.
- [LingQ](https://www.lingq.com/) - Reading and listening platform that tracks known and unknown words.
- [Lute](https://github.com/LuteOrg/lute-v3) - Open-source reading tool for learning languages through texts.
- [Language Reactor](https://www.languagereactor.com/) - Browser extension for learning from videos and subtitles.

## Vocabulary by Topic

- [WordMeadow](https://wordmeadow.app/) - Visual vocabulary theme cards across English, Spanish, French, Japanese, German, Korean, Italian, Chinese, Portuguese, and Hindi. Maintained by the WordMeadow team.
- [Drops](https://languagedrops.com/) - Short visual vocabulary practice organized by topics.
- [Digital Dialects](https://www.digitaldialects.com/) - Vocabulary games and topic sets for many languages.
- [Omniglot Phrases](https://omniglot.com/language/phrases/) - Phrase and vocabulary lists across languages.
- [BBC Languages Archive](https://www.bbc.co.uk/languages/) - Archived language learning material with useful beginner vocabulary.

## Spaced Repetition

- [FSRS](https://github.com/open-spaced-repetition/fsrs4anki) - Modern spaced-repetition scheduling algorithm used by many review tools.
- [AnkiDroid](https://github.com/ankidroid/Anki-Android) - Android client for Anki.
- [vocabsieve](https://github.com/FreeLanguageTools/vocabsieve) - Tool for adding looked-up words and sentences to Anki.
- [tatoeba-to-anki](https://github.com/Vuizur/tatoeba-to-anki) - Generate Anki decks using Tatoeba sentence translations.
- [fluentcards](https://github.com/katspaugh/fluentcards) - Convert Kindle lookups into Anki flashcards.

## Reading and Word Mining

- [KOReader](https://github.com/koreader/koreader) - E-reader app with dictionary support and multilingual reading workflows.
- [Yomitan](https://github.com/yomidevs/yomitan) - Browser extension for dictionary lookup and sentence mining, especially popular with Japanese learners.
- [mpvacious](https://github.com/Ajatt-Tools/mpvacious) - Create Anki cards from video subtitles.
- [asbplayer](https://github.com/killergerbah/asbplayer) - Browser-based media player for subtitle-based language learning.
- [LWT](https://github.com/HugoFara/lwt) - Self-hosted Learning With Texts implementation.

## Dictionaries and Pronunciation

- [Wiktionary](https://www.wiktionary.org/) - Collaborative multilingual dictionary.
- [Forvo](https://forvo.com/) - Native-speaker pronunciation recordings.
- [YouGlish](https://youglish.com/) - Hear words and phrases in real YouTube contexts.
- [WordReference](https://www.wordreference.com/) - Bilingual dictionaries and language forums.
- [Reverso Context](https://context.reverso.net/) - Translations and example sentences in context.
- [Linguee](https://www.linguee.com/) - Bilingual dictionary and translation examples from real documents.
- [DeepL Translator](https://www.deepl.com/translator) - High-quality translation for checking meaning and usage.

## Example Sentences and Corpora

- [Tatoeba](https://tatoeba.org/) - Open collection of translated example sentences.
- [YouGlish](https://youglish.com/) - Spoken examples from public videos.
- [Fraze.it](https://fraze.it/) - Sentence examples for English words.
- [SkELL](https://skell.sketchengine.eu/) - Corpus-based examples and collocations for English.
- [Leipzig Corpora Collection](https://wortschatz.uni-leipzig.de/en/download) - Downloadable corpora and word frequency data.
